Подробное описание

Заголовочный файл библиотеки

Аргументы с общим для всех функций значением:

Аргументы
mas(или же mas1 и mas2) - это указатели на обрабатывемые функцией массивы.
newmas- это указатель на массив в который будет записываться результат.
szEl- это размер обрабатываемого элемента массива.
size(или же size1 и size2 или newsize) - это число элементов в массиве.
iterate- указатель на функцию возвращающую адрес следующего элементаы, NULL - стоп-символ.

Значение суффиксов для функций:
New - означает что при работе результат будет записываться в новый массив.
Data - означает работу с любой структурой данных и принятие функции итерации как параметра.

Предупреждения
Память под новый массив должна быть выделена!

Значение суффиксов для макросов:
l(буква) - означает работу с одномерным массивом.

Предупреждения
Работа с двумерным массивом с помощью макросов с суффиксом l приведёт к сбою!

t - означает работу с двумерным массивом.
n - означает что при работе результат будет записываться в новый массив.

Предупреждения
Память под новый массив должна быть выделена!

i - означает работу с типом int.
f - означает работу с типом float.
d - означает работу с типом double.
e - означает работу с типом double в экспоненциальном виде.
c - означает работу с типом complex.
d - означает работу с либой структурой данных.
